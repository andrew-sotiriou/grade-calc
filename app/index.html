Let's start with how many grades you have to input per student

This is designed to take grades based on letters. You can only input these grades: A, A-, B+, B, B-, C+, C, D and F

For the weight of the _________ please enter those in the percentage. If it is 15% just put in 15. If its 7% put in 7. DO NOT PUT IT IN AS A DECIMAL!

If you are going to be entering more than 1 student for 1 class you can click the 'Weight Lock' button and then you wont accidently change the weight.
